Navegador web

Un navegador web (en inglés, web browser) es un software,aplicacion o programa que permite el acceso a la Web,interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos

La funcionalidad básica de un navegador web es permitir la visualizacion de documentos de texto,posiblemente con recusos multimedia incrustados
